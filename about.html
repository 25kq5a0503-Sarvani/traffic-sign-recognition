ℹ️ About This Project

Understanding the ML approach behind Traffic Sign Recognition

🎯 Project Goal

Build a web application that automatically detects and classifies traffic signs from images using a deep learning model. The system helps autonomous vehicles and driver-assistance systems recognize road signs in real-time.

🛠️ Tech Stack

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Python, Flask
  • ML Framework: TensorFlow / Keras
  • Model Type: CNN (Convolutional Neural Network)
  • Dataset: GTSRB (43 classes, 50k+ images)

🧠 What is CNN?

Convolutional Neural Network — the brain behind image recognition

A Convolutional Neural Network (CNN) is a type of deep learning model specially designed to process images. Just like our eyes scan an image and our brain identifies objects, a CNN scans the image layer by layer and learns patterns like edges, shapes, and colors.

In this project, the CNN looks at a traffic sign image and compares it to patterns it learned from 50,000+ training images to identify which of the 43 sign types it is.

🏗️ CNN Architecture Explained

What each layer does in our model

🔍 Conv2D Layer

Scans the image with small filters (3×3 pixels) to detect features like edges, curves, and colors. First layers detect simple features, deeper layers detect complex shapes like arrows and numbers.

📉 MaxPooling2D

Reduces the image size by half while keeping the most important information. This makes the model faster and prevents it from memorizing exact pixel positions.

🎲 Dropout

Randomly turns off some neurons during training. This prevents the model from memorizing training data (overfitting) and forces it to learn generalizable patterns.

🧮 Dense (Fully Connected)

Takes all the learned features and combines them to make the final prediction. The last Dense layer has 43 outputs (one for each sign class) with Softmax activation.

📊 Softmax Output

Converts raw scores into probabilities for all 43 classes. The class with the highest probability is the predicted sign. The confidence score shows how sure the model is.

⚡ Adam Optimizer

Automatically adjusts the model's learning rate during training to find the best weights faster. It's one of the most popular optimizers for deep learning.
